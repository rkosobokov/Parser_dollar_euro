# Парсер курса доллара и евро на Node.js + Selenium 
🤖 Проект настроен на непрерывную интеграцию (CI). Скрипт парсинга автоматически запускается по расписанию (cron) каждое утро в 09:00 UTC. Статус последних запусков доступен во вкладке Actions.
График и данные результата в папке reports.

## Установка на свой компьютер
1) Установить Node.js
2) cmd команда: npm install
3) Запустить, cmd команда: npx mocha test/yandex.spec.js --timeout 3600000
